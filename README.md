# Medict
Medict is a machine learning model integrated in a full stack web developent project,which predicts diseases on the basis of users input that are various symptoms one might be feeling at the moment 


## Tech Stack
- Django
- JavaScript
- Machine Learning
- SQLite
- Python
- GitHub
- CSS3
- HTML5
- Bootstrap
